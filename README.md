# Expense Splitter
A simple yet functional expense splitter built with Node.js and Express.

## Project Purpose
This project was created for three simple reasons:

1. **I was bored** - needed something fun to work on
2. **Learning JavaScript** - wanted to practice and improve my JavaScript skills
3. **Just for fun** - building things is enjoyable!

## Features
- Clean and modern web interface  
- Add and track shared expenses  
- Manage multiple participants  
- Automatic balance calculations  
- Smart settlement suggestions (who owes whom)  
- Beautiful gradient UI design  
- Fast and lightweight  
- Easy expense management with delete and clear options  

Made with ❤️ for learning and fun!
